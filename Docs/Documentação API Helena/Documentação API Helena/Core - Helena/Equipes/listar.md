# Listar Equipes

## Informações Gerais

**Título:** Listar

**Descrição:** Endpoint para listar departamentos/equipes do sistema.

## Detalhes Técnicos

**Método HTTP:** `GET`

**URL do Endpoint:** `https://api.wts.chat/core/v2/department`

**Versão da API:** v1.0

## Autenticação

Requerida via Header.

## Parâmetros

Nenhum parâmetro de query, path ou body especificado para este endpoint.

## Respostas

| Código | Descrição |
|--------|-----------|
| 200 | Sucesso — retorna a lista de equipes |
| 500 | Erro no servidor |

## Formato de Resposta

- **Content-Type:** application/json

## Linguagens Suportadas para Exemplos

Shell, Node, Ruby, PHP, Python
