# Obter Horários de Atendimento

## Informações Gerais

**Título:** Obter Horários de Atendimento

**Descrição:** Endpoint para recuperar as informações de horários de atendimento da empresa.

## Detalhes Técnicos

**Método HTTP:** `GET`

**URL do Endpoint:** `https://api.wts.chat/core/v1/company/officehours`

## Autenticação

Requerida via Header.

## Parâmetros

Nenhum parâmetro de query, path ou body documentado neste endpoint.

## Respostas

| Código | Descrição |
|--------|-----------|
| 200 | Retorna as informações de horários de atendimento da empresa |
| 500 | Erro no servidor |

## Formato de Resposta

- **Content-Type:** application/json

## Linguagens Suportadas para Exemplos

Shell, Node, Ruby, PHP, Python
