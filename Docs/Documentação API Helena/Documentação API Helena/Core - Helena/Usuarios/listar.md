# Listar Usuários

## Informações Gerais

**Título:** Listar

**Descrição:** Endpoint para listar todos os usuários (agentes) do sistema.

## Detalhes Técnicos

**Método HTTP:** `GET`

**URL do Endpoint:** `https://api.wts.chat/core/v1/agent`

**Versão da API:** v1.0

## Autenticação

Requerida via Header.

## Parâmetros

Nenhum parâmetro de query, path ou body especificado para este endpoint GET.

## Respostas

| Código | Descrição |
|--------|-----------|
| 200 | Retorna a lista de usuários/agentes |
| 500 | Erro no servidor |

## Formato de Resposta

- **Content-Type:** application/json

## Linguagens Suportadas para Exemplos

Shell, Node, Ruby, PHP, Python
